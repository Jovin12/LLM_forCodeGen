# LLM_forCodeGen
Creating an opensouce LLM, with prompt engineered code examples to help generate human defined code.
____________
If you are interested in using this please do,
First set up a fresh env, the python version used here is 11.14
Then install the requirements: 
```
pip install -r requirements.txt
```
Then run, the command to start the streamlit agent: 
```
streamlit run ./main.py
```
